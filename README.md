# Number Guesser

A simple number guessing game built using Python.

## How it Works
The program generates a random number between 1 and 100.
You have to guess the number and the program will tell you
if your guess is too high or too low until you get it right.

## Features
- Random number generation
- Tracks number of attempts
- Tells you if guess is too high or too low

## How to Run
```
python guess.py
```

## Example
```
Guess the number: 50
Too High
Guess the number: 25
Too Low
Guess the number: 37
Congratulations !!!!
You guessed the correct number in attempt: 3
```
